# :runner: How to run this project

```bash
# Clone this repository
$ git clone https://github.com/philipe-vieira/proffy.git

# Install dependencies
$ yarn

# Running API
$ yarn api:run
# Or, in development mode
$ yarn api:run-dev

# Running Web App
$ yarn web

# Running Mobile App
$ yarn mobile
```
